Your code needs to search \textbf{breeds} for "Afghan Hound" \& return its location in the array, i.e., index. If "Afghan Hound" is not in \textbf{breeds}, return -1.

\begin{verbatim}

// 1

// Make sure to test if Afghan Hound is not found
\end{verbatim}
